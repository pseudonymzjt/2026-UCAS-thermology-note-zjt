# 2026-UCAS-thermology-note-zjt

这是一个用于整理和分享 **中国科学院大学（UCAS）2026学年热学课程** 个人笔记的开源仓库。

## 📌 课程与笔记信息

- **课程名称**：热学 (Thermology)
- **授课学校**：中国科学院大学 (UCAS)
- **授课教师**：傅金林老师
- **参考教材**：《热学》（刘玉鑫编著，北京大学出版社）
- **学期时间**：2026年春季学期
- **笔记整理**：[pseudonymzjt](https://github.com/pseudonymzjt)

## 📝 笔记主要内容

本笔记根据授课大纲逐步整理，主要涵盖以下知识模块：

1. **导论与温度**：热力学系统的状态描述、平衡态、温度与热力学第零定律。
2. **气体分子动理论**：平衡态的统计分布规律、麦克斯韦速率分布、能量均分定理、理想气体的压强与温度微观本质。
3. **热力学第一定律**：热量、内能、功、第一定律的应用、循环过程与卡诺循环。
4. **热力学第二定律**：开尔文与克劳修斯表述、卡诺定理、熵与熵增加原理、热力学第二定律的微观意义。
5. **输运过程**：黏滞、热传导、扩散现象的微观物理机制。
6. **相变与真实气体**：气液相变、范德瓦耳斯方程、相平衡与克拉珀龙方程。
